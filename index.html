自傳

我是曾暐傑，我想要應徵DEXON 校園大使。我對教學與分享知識充滿熱忱，擅長自學程式與分享知識，此外也擅長撰寫心得程式的技術筆記。


一年左右的Web專案開發經驗
為了要製作專題，自學基礎前後端能力(PHP FrameWork- Laravel)開發旅遊分享平台TraEasy，也在途中Google認證夥伴-業展資訊學習Google Api串接以及後台製作。

Web相關自學資歷約莫一年左右，開發過的專案如前面履歷所述。 
熟悉 Web並開始學習區塊鏈知識

網頁功能邏輯在業展資訊中學到很多，網頁開發能力都是且戰且走的學習，先有了具體的目標，然後再學相關的知識。希望在學習的這條路上可以撰寫技術相關文章幫助更多的人，少撞一些牆。

 除了不斷精進Web前後端的知識外，也開始參加了台灣南北部大大小小的Meetup和將這些資訊累積撰寫成文章，。

熟悉 Git 並可流暢的進行多人協作

初學Web時受困於不善git操作，當時也到處詢問畢業後的學長姐，最終在個人開發上功能十分熟悉。由於是自學的緣故，尚未有多人協作的經驗，但是相信自己能快速上手。

舉辦過大型活動以及在外分享經驗

大學時期及高中時期皆為熱舞社公關，當時有和夥伴共同舉辦過Taoyuan New Star Session的大型街舞活動，也在最近有到高科大其他校區以及好想工作室分享區塊鏈知識，希望能讓更多人接觸這個產業及文化。

使用 API 與第三方服務

目前串過的第三方服務是用 devise + GoogleAuth來實作Google登入以及Direcotry api將Google Auth做分層級的batch指令去完成CRUD Function。

能夠透過英文吸收新知

大學時就已經有讀原文書的習慣，也喜歡看Medium英文技術文章，當程式有bug也喜歡上stackoverflow or github上面找尋自己需要的程式片段加以解析。

完成任何程式相關的 MOOC 課程

在大四接碩一時，由於學校課程教的知識過少，鮮少實作專案。平常時，會觀看Hahow以及Udemy上的教學影片，並利用線上的workshop練習，目前看過的教學影片也不外乎就是Web前後端以及Blockchain簡易實作。

結語

我是Panda，我有能力開發web及blockchain相關Application，遇到的問題也能透過恰當的關鍵字自行google找到可行的解法。並自學使用API與第三方服務銜接整合。除了維護開發與內部使用的 API，有著優秀的文字駕御能力來撰寫文件，並且會寫筆記記錄工作上所遇到的問題與解決方案，日後新進來的夥伴更能閱讀相關筆記，協助當事人更快速上手工作，在大學時期也是熱舞社公關，有舉辦活動相關經驗，也希望未來能在Dexon推廣上將這些技能用上。
